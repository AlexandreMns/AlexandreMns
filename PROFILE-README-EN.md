## 🚀 Featured Projects

### 🤖 SNCT AI Chatbot (RAG) — SNCT Internship, Luxembourg
AI chatbot for the SNCT institutional website (a vehicle technical inspection company), built as a custom WordPress plugin.

- **What it is:** a multilingual (EN/DE/FR) conversational assistant that answers based on real website content — the inspection sanctions/defects catalog, pricing, FAQ, and other institutional pages — instead of relying solely on the model's general knowledge.
- **RAG architecture:** embedding pipeline via **Azure OpenAI**, vector storage and search via **Azure AI Search**, and final response generation grounded in the retrieved context.
- **Automated data pipeline:** Node.js scrapers (Playwright + Cheerio for interactive content, WordPress REST API for static pages), orchestrated via **GitHub Actions** on a scheduled cadence, keeping the knowledge base continuously up to date with no manual intervention.
- **Technical highlights:** automatic language detection based on the current page (Polylang integration), Markdown-to-HTML rendering in the chat widget, security hardening against prompt injection/jailbreak attempts, and cost optimization via reduced embedding dimensions to fit free-tier Azure AI Search limits.

### 🧵 Fashable — Full-Stack AI Fashion Platform
Full-stack AI-powered fashion platform, developed in a team environment within an NX monorepo.

- **Stack:** Node.js, TypeScript, NX Monorepo.
- **What it is:** a platform featuring AI-powered visual image search, an authentication system, and a credits/ticketing system for users.
- **Technical highlights:** implemented vector-based image search with **Azure Cognitive Search**, secure authentication with **Keycloak/JWT**, and built Express routers for an image suggestion service (`SuggestImageService`).
